# Bot de Restaurante para Discord

Bot desenvolvido em Python para automatizar pedidos dentro do Discord. Ele exibe o cardápio com botões interativos, permite adicionar itens ao carrinho, escolher tamanhos/sabores, finalizar pedido (nome, endereço e pagamento) e avaliar o atendimento com estrelas.

## Funcionalidades

* Menu interativo com categorias e botões
* Carrinho de compras com itens e total
* Fluxo completo de checkout
* Mensagem final de pedido + avaliação por estrelas
* Totalmente configurado via `.env`

## Como executar

1. Instale dependências (`pip install -r requirements.txt`)
2. Crie o arquivo `.env` com: `TOKEN=seu_token_aqui`
3. Execute: `python bot.py`

## Tecnologias

Python • Discord.py • python-dotenv
